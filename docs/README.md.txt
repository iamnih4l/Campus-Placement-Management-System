# Campus Placement Management System

## Team Members
- Mohammed Nihal

## Technologies Used
- Java (Swing) - Frontend
- MySQL - Backend Database
- JDBC (MySQL Connector/J)

## Database
- SQL file: sql/campus_placement_full_100rows.sql
- Tables:
  - students (100 rows)
  - companies (100 rows)
  - jobs (100 rows)
  - applications (100 rows)
  - interviews (100 rows)
  - users (10 rows)

## Features (Planned)
- Student management
- Company and job posting management
- Job applications
- Interview scheduling
- Login system
- Placement dashboard

## Requirements
- JDK 17+
- MySQL 8.x
- MySQL Connector/J driver
